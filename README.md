# Student Management System (Python CLI)

## Description

A simple command-line application to manage student records. It allows adding, viewing, updating, and deleting student information.

## Features

* Add student records
* View all students
* Update existing student details
* Delete student records

## How to Run

```bash
python student_tracker.py
```

## Concepts Used

* Dictionaries
* Functions
* Conditional statements
* Loops
